# About This Site
This site's designs and technologies were not designed by me. They were designed by [Renovamen](https://github.com/Renovamen), and it was absolutely mind blowing for a portfolio site, so I cloned and customized it, shamelessly.

This site is inspired by macOS [Big Sur](https://www.apple.com/in/macos/big-sur/) and [Catalina](https://www.apple.com/bw/macos/catalina/), developed using [React](https://reactjs.org/), [React Redux](https://react-redux.js.org/) and [Tailwind CSS](https://tailwindcss.com/), [Github Pages](https://pages.github.com/). Some of the icons are generated using [sindresorhus/file-icon-cli](https://github.com/sindresorhus/file-icon-cli).

The site code is hosted [here](https://github.com/absozero/absozero.github.io).
